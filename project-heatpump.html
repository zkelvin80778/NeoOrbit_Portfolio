3D-Printed Heat Pump

Habitat Temperature Regulation for Martian Missions

Project Overview

This project tackles the unique challenge of habitat temperature regulation for future Martian missions. Utilizing thermoelectric generators, 3D-printed components, and open-source control systems, it aims to provide practical solutions for off-world sustainability.

Technical Details

The system is designed around a core of thermoelectric generators (TEGs), also known as Peltier modules, which can heat or cool by applying an electric current.

  • Core Technology: Thermoelectric Generators (TEGs) / Peltier Modules
  • Manufacturing: Key components 3D-printed from regolith-simulant composites.
  • Control System: Arduino-based open-source controller.
  • Current Status: TRL 3 (Proof-of-concept validated)
